### Указатели и референции
--------------------------------------------------
Теоретични задачи:
---
1. Какво ще се отпечата на екрана?

```cpp
#include <iostream>

int main() {
    int* p;
    *p = 10;
    std::cout << *p << std::endl;
    return 0;
}

```

2. Какво ще се отпечата на екрана?

```cpp
#include <iostream>

void addFive(int& x) {
    x += 5;
}

int main() {
    int num = 10;
    addFive(num);
    std::cout << num << std::endl;
}
```

3. Какво ще се отпечата на екрана?

```cpp
#include <iostream>
void swap(int* a, int& b) {
    int temp = *a;
    *a = b;
    b = temp;
}

int main() {
    int x = 3, y = 7;
    swap(&x, y);
    std::cout << x << " " << y << std::endl;
    return 0;
}
```
