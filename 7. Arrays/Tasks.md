### Масиви
--------------------------------------------------
Теоретични задачи:
---

1. Какво ще изведе програмата?
```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5] = {1, 2, 3, 4, 5};
    cout << a[2] << endl;
    cout << a[0] + a[4] << endl;
}
```

<details>
 <summary>Отговор: </summary>
Кодът отпечатва стойността на третия елемент a[2], която е 3, и сумата на първия и последния елемент a[0] + a[4], която е 1 + 5 = 6.
</details>

2. Има ли грешка?
   
```cpp
int main() {
    int arr[3];
    arr[0] = 5;
    arr[1] = 10;
    arr[3] = 15;
}
```

<details>
 <summary>Отговор: </summary>
Да, има грешка – arr[3] = 15; е извън границите на масива с размер 3 (валидни индекси са 0, 1, 2).
Това води до undefined behaviour.
</details>

3. Какво ще се изведе?
   
```cpp
int main() {
    int x[5] = {2};
    cout << x[0] << " " << x[1];
}
```

<details>
 <summary>Отговор: </summary>
Масивът x[5] се инициализира с {2}, което задава само първия елемент, а останалите се нулират по подразбиране.
Затова изходът ще бъде:
	
```
2 0
```
</details>

4.Какво ще се изведе и как изглежда масивът във всеки момент?

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5] = {3, 1, 4, 1, 5};
    a[2] = a[0] + a[4];
    a[4] = a[1];
    cout << a[2] << " " << a[4];
}
```

<details>
 <summary>Отговор: </summary>
Кодът първо променя a[2] на 3 + 5 = 8, после a[4] на 1, така че масивът става {3, 1, 8, 1, 1}.
Изходът ще бъде:

```
8 1
```
</details>

5. Валиден ли е следният код?

```cpp
int a[3] = {1,2,3};
int b[3];
b = a;
```

<details>
 <summary>Отговор: </summary>
НЕ!
</details>
