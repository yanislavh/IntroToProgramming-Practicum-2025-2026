### Многомерни масиви
--------------------------------------------------
Теоретични задачи:
---

1. Какво ще изведе програмата?
```cpp
#include <iostream>
using namespace std;

int main() {
    int a[2][3] = {{1, 2}, {3}};
    cout << a[0][2] << " " << a[1][1] << endl;
}
```
2. Валиден ли е кодът?
```cpp
int a[2][2] = {{1,2},{3,4}};
int b[2][2];
b = a;
```

3. Какво ще изведе програмата?
```cpp
#include <iostream>
using namespace std;

int main() {
    int a[2][3] = {{1,2,3},{4,5,6}};
    int temp = a[0][0];
    a[0][0] = a[1][2];
    a[1][2] = temp;
    cout << a[0][0] << " " << a[1][2] << endl;
}
```

</br>

```

```

```
9 + 5 + 1 = 15

2 + 9 + 4 = 15
7 + 5 + 3 = 15
6 + 1 + 8 = 15
Сумите на двата диагонала са:
2 + 5 + 8 = 15
6 + 5 + 4 = 15
```
